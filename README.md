Wercker step for box : olivermt/grails 

Usage
-----------
    box: olivermt/grails
    build:
      steps:
        - olivermt/grails-step:
            version: 2.5.0
            options: help
Display help : similar to command "grails help"

You can make every command after "option: your grails command"

Only 2.5.0 supported at the moment
